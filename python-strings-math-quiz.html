The big ideas.

Refresh on the five concepts before the quiz.

Concept 01

Formatted strings (f-strings)

Put an f before a string to plug variables right in using { }.

first = "John"
last = "Smith"
message = f"Hi, I am {first} {last}"
print(message)
# Hi, I am John Smith
Concept 02

String methods (built-in tools)

Strings come with useful methods. Use a dot to call them.

course = "Python for Beginners"

print(len(course))           # 20
print(course.upper())       # PYTHON FOR BEGINNERS
print(course.lower())       # python for beginners
print(course.find("P"))     # 0  (position)
print(course.replace("P", "-"))   # -ython for Beginners
print("Python" in course)  # True

Important: upper() and lower() do NOT change the original string — they return a new one.

Concept 03

Arithmetic operators

The basic math operators in Python:

• + add — 10 + 3 = 13
• - subtract — 10 - 3 = 7
• * multiply — 10 * 3 = 30
• / divide — 10 / 3 = 3.333 (always float)
• // integer divide — 10 // 3 = 3 (drops decimal)
• % remainder — 10 % 3 = 1 (what's left over)
• ** exponent — 10 ** 3 = 1000 (power)

Concept 04

Operator precedence

The order Python does math: 1) ** first, 2) * / // % next, 3) + - last. Use ( ) to force a different order.

print(10 + 3 * 2)        # 16  (multiply first)
print((10 + 3) * 2)      # 26  (parentheses first)
Concept 05

Math functions

Built-in: round() rounds, abs() gives absolute value (always positive). For more, import math.

print(round(2.9))         # 3
print(abs(-2.9))         # 2.9

import math
print(math.ceil(2.2))    # 3 (rounds up)
print(math.floor(2.9))   # 2 (rounds down)
