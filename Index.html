Lesson · Python Basics

Talking to the computer.

Python is how you tell a computer what to do. Here are the four things you need to know for this quiz.

Concept 01

Variables store stuff

A variable is a labeled box. You put a value in, then use the label later.

name = "Alex"
age = 12
is_cool = True
Concept 02

print() shows output

print() displays whatever you put inside the parentheses.

print("Hello, world!")   # Hello, world!
print(5 + 3)              # 8
print(name)               # Alex
Concept 03

Data types

Python knows different kinds of data:

• str — text in quotes: "hello"
• int — whole numbers: 42
• float — decimals: 3.14
• bool — True or False

Concept 04

if / else makes decisions

Code can choose what to do based on a condition.

age = 12
if age >= 13:
    print("Teenager")
else:
    print("Not yet")
# Prints: Not yet
