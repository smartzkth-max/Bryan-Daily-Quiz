Python Loops & Lists.

Lesson · Loops & Lists

Groups and repeats.

Yesterday you learned variables. Today: storing lots of things in one variable, then doing stuff to each one.

Concept 01

Lists hold many things

A list is like a row of boxes, each holding a value. Use square brackets [ ].

fruits = ["apple", "banana", "cherry"]
numbers = [10, 20, 30, 40]
Concept 02

Index starts at 0

To get one item, use its position number inside [ ]. The first item is 0, not 1.

fruits = ["apple", "banana", "cherry"]
print(fruits[0])   # apple
print(fruits[1])   # banana
print(fruits[2])   # cherry
Concept 03

len() counts items

len() tells you how many items are in a list.

fruits = ["apple", "banana", "cherry"]
print(len(fruits))   # 3
Concept 04

for loops repeat actions

A for loop does something to each item in a list.

fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(fruit)
# Prints:
# apple
# banana
# cherry
Concept 05

range() makes number sequences

range(5) gives you 0, 1, 2, 3, 4 (stops before 5). Great for counting loops.

for i in range(3):
    print(i)
# Prints: 0, then 1, then 2
Concept 06

append() adds to a list

To add an item to the end of a list, use .append().

fruits = ["apple", "banana"]
fruits.append("cherry")
print(fruits)
# ["apple", "banana", "cherry"]

Take your time with these — loops are powerful once you get them. Ready?
