Making decisions.

Refresh on the four concepts before the quiz.

Concept 01

If / elif / else

Tells Python to do different things based on a condition. Indentation matters!

temp = 25

if temp > 30:
    print("It's hot")
elif temp > 20:
    print("It's warm")
else:
    print("It's cold")
# Prints: It's warm

elif means "else if" — only checked if the previous condition was False.

Concept 02

Comparison operators

How to compare two values. They always give back True or False.

• == equals (note: TWO equal signs!)
• != not equal
• > greater than
• < less than
• >= greater than or equal
• <= less than or equal

Important: = assigns a value, == compares values. Don't mix them up.

Concept 03

Logical operators: and, or, not

Combine multiple conditions.

age = 15

# 'and' = ALL conditions must be True
if age >= 13 and age <= 19:
    print("Teenager")

# 'or' = AT LEAST ONE condition must be True
if age < 5 or age > 65:
    print("Free entry")

# 'not' = flips True to False
is_hot = False
if not is_hot:
    print("Bring a jacket")
Concept 04

Weight converter (putting it together)

This is the kind of small program you can build with what you know now.

weight = int(input("Weight: "))
unit = input("(L)bs or (K)g: ")

if unit.upper() == "L":
    converted = weight * 0.45
    print(f"You are {converted} kilos")
else:
    converted = weight / 0.45
    print(f"You are {converted} pounds")

This combines: input(), int(), math, if/else, comparison, string method, f-string. Almost everything you've learned!
